# 🍵 Matcha Time - React Admin Dashboard

Welcome to the **Matcha Time Admin Dashboard**! This is a modern, responsive, and fully functional React-based admin panel designed for a premium Matcha cafe e-commerce platform.

## ✨ Features

- **📊 Dashboard:** Overview of sales, revenue, and active orders using dynamic charts (Recharts).
- **🛍️ Products Management:** View, add, and manage matcha products with image uploads and stock status.
- **📦 Orders Management:** Track customer orders, update statuses (Pending, Processing, Delivered), with built-in pagination.
- **📝 Content Management System (CMS):** Dynamically control frontend content:
  - Header & Hero (Brand titles, background images)
  - About Us section (Story and imagery)
  - Testimonials (Customer reviews)
  - Visit Our Shop (Location, Maps, Operating Hours)
- **🖼️ Gallery:** Drag-and-drop image upload interface for managing store media.
- **💬 Messages:** Inbox-style UI to read and reply to customer inquiries.
- **⚙️ Settings:** Manage admin profile, store currency/timezone, notifications, and security.

## 🛠️ Technology Stack

- **React 18** (Vite)
- **Tailwind CSS** (Styling & Responsive Design)
- **Lucide React** (Modern Icons)
- **Recharts** (Data Visualization)
- **React Router DOM** (Navigation)

## 🚀 Getting Started

To run this Admin Panel locally on your machine:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. Open your browser and visit the local server address (usually `http://localhost:5173`).
